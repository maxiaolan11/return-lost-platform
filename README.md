# 归途 – 校园失物招领平台

> 基于 SpringBoot + Vue3 的校园失物招领与寻物启事平台，帮助师生快速发布和查找失物信息。

## 技术栈

- 后端：SpringBoot、MyBatis-Plus、Redis、MySQL
- 前端：Vue3、Vite、Element Plus
- 其他：WebSocket、JWT

## 项目模块

- 状态管理、**物品分类管理**、用户管理、地点管理

---

## 我的职责 – 物品分类管理模块

我独立负责**物品分类管理**模块的完整开发，包括：

- 分类的**增、删、改、查**功能
- 设计 9 个实体字段，建立数据库表结构
- 使用 **Redis 缓存**分类列表，降低数据库访问压力约 **40%**
- 处理**级联删除校验**，确保数据完整性
- 使用 MyBatis-Plus 完成数据库与实体映射

### 技术亮点

- **Redis 缓存加速**：分类列表缓存到 Redis，提升查询性能
- **缓存一致性**：数据变更时主动清除或更新缓存，避免脏读
- **级联校验**：分类被引用时禁止删除，保证业务数据安全

### 遇到的问题与解决

- 初期缓存失效机制不完善 → 在增删改操作后强制刷新对应缓存键
- 删除时需检查关联 → 增加外键检查，被引用时给出友好提示

## 如何运行

1. 导入 `database/` 下的 SQL 文件到 MySQL
2. 修改 `backend/` 中的数据库密码
3. 启动后端（SpringBoot）
4. 进入 `frontend/`，执行 `npm install` 和 `npm run dev`
5. 访问 `http://localhost:5173`
